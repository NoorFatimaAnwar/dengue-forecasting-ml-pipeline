# Dengue Fever Dataset Information

The dataset used in this project is private and cannot be uploaded publicly.

To reproduce the project:

1. Download the dataset from this link:
   <[Dataset link here](https://www.drivendata.org/competitions/44/dengai-predicting-disease-spread/data/)>

2. Create a folder named `data/` in your local machine or runtime.

3. Place these files inside `data/`:

   - dengue_features_train.csv
   - dengue_labels_train.csv
   - dengue_features_test.csv

4. Open the notebooks from the `/notebooks/` folder and upload these files if needed.
